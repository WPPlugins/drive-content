== Description ==

This plugin will tie into the [AutoNetTv Drive](http://drive.autonettv.com) system to allow you to add your Drive Content Blocks to your wordpress website using the Editor.

This plugin will add a button in your editor that when clicked will launch a window that will allow you to log into your Drive Account. Once logged in it will load all of the available Drive Content blocks that you have. You will then be able to select a block and a size and it will insert a short code into your page. This short code will then be translated when the page is loaded into the Drive Content Block.

== Installation ==

1. You can download the zip and upload the extracted folder to your plugins folder.
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Do I have to have an account with AutoNetTv Drive to use this plugin? =

Yes. This plugin works by loading the Drive Content Blocks associated with your Drive Account.

= How do i sign up for an AutoNetTv Drive account? =

You can lean more about AutoNetTv Drive and how to sign up for an account on our website at: [driveContent](http://www.autonettv.com/products/drive-content/)
